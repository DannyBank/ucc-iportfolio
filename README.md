Personal Portfolio Website

A modern, responsive portfolio website showcasing the skills, projects, and professional profile of Daniel Bucknor-Ankrah, a website designer and software developer.

Overview

This portfolio was built as a lightweight single-page application that dynamically loads page content without requiring full page refreshes. It provides visitors with an easy way to learn about my background, view my projects, and get in touch.

Features
Responsive design for desktop, tablet, and mobile devices
Dynamic page loading using JavaScript
Light and Dark theme toggle with preference saved in the browser
Smooth scrolling and animations
Portfolio/projects showcase
About Me section
Contact section
Social media integration
Clean and modern UI

Getting Started
Clone the repository
git clone https://github.com/DannyBank/ucc-iportfolio.git
Open the project

Simply open index.html in your browser.

Alternatively, you can use a local development server for the best experience.

For example: go to the root directory and run "http-server"
